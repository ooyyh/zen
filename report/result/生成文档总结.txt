                         🛠️ 图表生成方法
================================================================================

【方法一】在线生成（推荐，无需安装）
1. 打开 http://www.plantuml.com/plantuml/
2. 打开 d:/Project/IdeaProjects/zen/report/template/charts/*.puml 文件
3. 复制内容到在线编辑器左侧
4. 右侧自动显示图表，右键保存为PNG

【方法二】本地生成（需要安装PlantUML）
1. 下载 https://plantuml.com/download 或 apt-get install plantuml
2. 在charts目录执行：plantuml -tpng *.puml
3. 在同目录生成PNG文件

【方法三】如果不想生成图表
- 在Word中手动绘制系统设计图
- 或使用Visio、Lucidchart等工具绘制
- 或从现有的系统文档中导出
