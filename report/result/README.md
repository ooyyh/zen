## 🚀 使用步骤

### 第一步：生成图表
```bash
# 使用PlantUML在线工具或本地工具生成图表
# 访问 http://www.plantuml.com/plantuml/ 或使用本地PlantUML
# 将 charts/*.puml 文件转换为 PNG/SVG 格式
```

### 第二步：插入图表到Word
1. 打开生成的Word文档
2. 找到 `[图表名称.puml]` 标记
3. 删除标记，将生成的图片插入
4. 调整图片大小和位置

### 第三步：完善个人信息
1. 填写学生姓名、学号、班级等个人信息
2. 填写指导教师名称和工作单位
3. 填写时间日期信息
4. 根据需要调整任务书和开题报告中的内容

### 第四步：最终审查
- 检查所有内容的准确性
- 确保图表正确插入
- 验证排版和格式
- 打印或导出为PDF
